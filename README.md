## 🚀 热门项目

### [Linux TCP Socket C++](https://github.com/desyang-hub/linux-tcp-socket-cpp)
Linux下的C++ Socket网络编程实战项目，从基础TCP通信到高性能网络编程。
- 🔌 基础TCP客户端/服务器实现
- ⚡ IO多路复用技术（Epoll）
- 🧵 多进程与多线程并发模型
- 🏗️ 线程池设计与实现
- 🚀 轻量级RPC框架（基于Protobuf）
- 📚 完整的中文教程与代码示例


### [Mini-rpc-cpp](https://github.com/desyang-hub/mini-rpc-cpp)
基于 C++ 和 Protobuf 的轻量级 RPC 框架，支持高性能网络通信。
- ⚡ 基于 Epoll 的高性能 I/O 多路复用
- 🔄 支持多线程并发处理请求
- 📦 集成 Protobuf 序列化，实现高效数据传输
- 🎯 提供完整的 RPC 服务提供者和客户端通道实现


### [YOLOv8 ONNX C++](https://github.com/desyang-hub/yolo-onnx-cpp)
基于 ONNX Runtime 和 OpenCV 的 YOLOv8 目标检测 C++ 实现，支持实时视频流处理。
- ⭐ 多线程流水线设计
- 🎯 完整的 YOLOv8n 模型推理支持
- 🐳 Docker 部署方案

### [Design Patterns in C++](https://github.com/desyang-hub/design-patterns-cpp)
C++ 设计模式实战教程，精选 7 种最常用、最核心的设计模式，帮助开发者快速掌握设计模式的精髓。
- 📚 包含单例、工厂、观察者、策略、装饰器、适配器、代理等经典模式
- 💡 每种模式配有详细的代码示例和应用场景说明
- 🎯 聚焦实战应用，解决 80% 以上的常见架构问题
- 🔧 完整的 C++11/14 实现示例，包含测试用例
